Solutions

围绕企业知识流转设计的解决方案

知识库检索中心

统一接入制度文档、技术 Wiki、项目复盘和 FAQ，用关键词与向量召回提升查找效率。

业务 Agent 助手

面向客服、人力、研发和法务场景配置专属 Agent，自动拆解任务并调用工具。

支持企业内部模型、权限、审计和知识库隔离，满足数据安全与团队协作需求。

Workflow

从企业文档到智能答案的执行链路

  1. 01理解问题

    识别任务类型、知识库范围和需要调用的工具。

  2. 02融合检索

    关键词召回与向量召回共同进入候选池，再重排为可用证据。

  3. 03工具执行

    按 Agent Loop 多轮调用知识库、数据库、文件和邮件工具。

  4. 04生成引用

    输出结论、依据、步骤和风险提示，并附带来源。
