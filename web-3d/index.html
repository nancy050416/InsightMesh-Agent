Product

把企业资料变成可追溯的行动答案

混合召回

结合关键词检索、向量检索和规则过滤，兼顾接口名、制度编号与语义表达。

Agent 决策

先规划，再调用知识库、数据库、文件或邮件工具，复杂任务按步骤完成。

来源引用

答案绑定文档标题、章节路径和片段编号，降低无依据回答风险。

Workflow

从问题到答案的执行链路

  1. 01理解问题

    识别任务类型、知识库范围和需要调用的工具。

  2. 02融合检索

    关键词召回与向量召回共同进入候选池，再重排为可用证据。

  3. 03工具执行

    按 Agent Loop 多轮调用知识库、数据库、文件和邮件工具。

  4. 04生成引用

    输出结论、依据、步骤和风险提示，并附带来源。
