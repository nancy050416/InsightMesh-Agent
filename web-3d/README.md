# InsightMesh AI 3D 官网

这是“应用式能力测评”的 Three.js 作品。页面以企业知识库混合检索 Agent 为公司产品，使用 Three.js 构建沉浸式 3D 知识网络，展示文档节点、语义连接、Agent 决策核心和检索路径。

## 运行

```bash
npm install
npm run dev
```

访问 `http://localhost:5173`。

## 构建

```bash
npm run build
```

构建产物位于 `dist/`。

## 视觉设计

- 3D 场景：中心 Agent 核心、环形推理轨道、文档节点、语义连接线和背景粒子。
- 交互：鼠标移动时场景产生轻微视差，节点和连接持续动态更新。
- 页面结构：首页首屏、产品能力、工作流、技术栈。
- 主题：克制的企业产品视觉，用墨绿、青绿、琥珀、珊瑚色区分不同信息层级。
