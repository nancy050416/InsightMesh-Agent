# InsightMesh AI 3D 官网

这是“应用式能力测评”的 Three.js 多页面公司官网作品。页面以虚拟公司 InsightMesh AI 智能检索科技公司为主体，展示公司定位、解决方案、典型客户场景、技术栈和联系方式，并使用 Three.js 构建沉浸式 3D 知识网络。

## 运行

```bash
npm install
npm run dev
```

访问 `http://localhost:5173`。

## 构建

```bash
npm run build
```

构建产物位于 `dist/`。

## 视觉设计

- 3D 场景：中心 Agent 核心、环形推理轨道、文档节点、语义连接线和背景粒子。
- 交互：鼠标移动时场景产生轻微视差，节点和连接持续动态更新。
- 页面结构：首页、公司、解决方案、客户案例、技术架构、联系页面。
- 导航行为：按钮和导航进入独立页面，不使用单页锚点滚动。
- 主题：克制的企业产品视觉，用墨绿、青绿、琥珀、珊瑚色区分不同信息层级。
